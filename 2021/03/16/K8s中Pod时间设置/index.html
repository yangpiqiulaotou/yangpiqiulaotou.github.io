发表于 2021-03-16 更新于 2022-04-17 分类于 Kubernetes

1、 时间介绍

1.1 GMT

​ 格林威治时间。即格林威治所在地的标准时间。

1.2 UTC

​ 世界标准时间（即 GMT）。即：UTC = GMT + 0。容器中的时间就是 UTC。

1.3 CST

​ 中国标准时间。CST = GMT + 8。

1.4 EST

​ 东部时间。EST = GMT - 5。如果当前机子所在时区是纽约，那么日期显示就是 EST。
